# Account lockout after failed attempts

## Preconditions

- User account exists
- No previous failed attempts

## Steps

1. Navigate to login page
2. Enter valid email address
3. Enter incorrect password
4. Click "Login" button
5. Repeat steps 3-4 five times

## Expected Result

- After 5th failed attempt, account is locked
- Error message "Account locked due to too many failed attempts" displayed
- Lockout lasts 15 minutes
- Correct password rejected during lockout
