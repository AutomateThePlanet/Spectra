# Verify notification preferences take effect immediately after change

## Preconditions

Citizen is logged in with all notification channels enabled by default.

## Steps

1. Navigate to notification preferences page
2. Disable email notifications for a specific category (e.g., 'Application status change')
3. Save the preference changes
4. Trigger an application status change event immediately
5. Check email inbox and in-app notifications
6. Verify email is NOT received but in-app notification is still delivered

## Expected Result

Preference changes take effect immediately. No email notification is sent for the disabled category. In-app notification is still delivered as it cannot be disabled. Notifications already queued before the preference change are still sent.

## Test Data

Valid citizen account with active application
