# Verify SMS notification truncation when message exceeds character limit

## Preconditions

Citizen has SMS notifications enabled. A notification event is triggered that generates SMS content exceeding 160 characters (Latin) or 70 characters (Cyrillic).

## Steps

1. Configure a test notification template with content exceeding SMS character limits (>160 Latin or >70 Cyrillic)
2. Trigger a notification event that uses this template (e.g., payment received with long service name)
3. Check the SMS received on the citizen's registered phone number
4. Verify the SMS content is truncated
5. Verify the truncated message ends with '...' and includes a link to the full notification in the portal

## Expected Result

SMS message is truncated to 160 characters (Latin) or 70 characters (Cyrillic). The truncated message ends with '...' followed by a link to view the full notification in the portal.

## Test Data

Notification template with content exceeding character limits
