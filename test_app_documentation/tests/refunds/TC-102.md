# Partial refund for expedited fee only

## Preconditions

Application was paid with expedited fee but will be processed as standard service

## Steps

1. Log in as administrator with payments.refund permission
2. Navigate to application with expedited fee payment (e.g., Certificate of Birth: 5.00 BGN base + 15.00 BGN expedited = 20.00 BGN total)
3. Select 'Process Partial Refund'
4. Enter refund amount: 15.00 BGN
5. Enter reason: 'Expedited processing not available, refunding expedited fee'
6. Verify system shows breakdown: Original: 20.00 BGN, Refund: 15.00 BGN, Retained: 5.00 BGN
7. Confirm partial refund
8. Verify payment status shows 'Partially Refunded'

## Expected Result

Partial refund of 15.00 BGN is processed. Payment record shows original amount (20.00 BGN), refunded amount (15.00 BGN), and net amount (5.00 BGN). Citizen receives email with partial refund details. Application continues processing with standard fee.

## Test Data

Card payment: 20.00 BGN (5.00 base + 15.00 expedited), Card ending in 5678
