# Refund card payment after application rejection

## Preconditions

Citizen has submitted an application with card payment (Visa/Mastercard) that has been rejected by administration

## Steps

1. Log in as administrator with payments.refund permission
2. Navigate to the rejected application
3. Click 'Process Refund' button
4. Verify refund method is set to 'Original Card' (last 4 digits displayed)
5. Confirm refund amount matches original payment
6. Submit refund request
7. Verify refund status changes to 'Processing'
8. Verify citizen receives email notification about refund

## Expected Result

Refund is processed back to the original card. System displays message 'Refund will be processed within 5-10 business days'. Payment status changes to REFUNDED. Refund transaction is logged in audit trail.

## Test Data

Application with card payment: 50.00 BGN, Card ending in 1234
