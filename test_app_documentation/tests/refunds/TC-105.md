# Reject cash refund without original receipt

## Preconditions

Application was paid in cash at government office and is now eligible for refund

## Steps

1. Citizen visits government office to request cash refund
2. Clerk logs in with payments.refund permission
3. Clerk navigates to the application requiring refund
4. Clerk selects 'Process Cash Refund'
5. System prompts for original receipt number
6. Clerk attempts to process refund without entering receipt number
7. Clerk clicks 'Submit Refund'
8. Observe system validation

## Expected Result

System rejects refund request with error: 'Original receipt number is required for cash refunds'. Refund cannot be processed. Clerk must verify citizen has original receipt and enter receipt number to proceed. Payment status remains unchanged.
