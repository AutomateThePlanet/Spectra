# Refund voucher payment as credit note

## Preconditions

Citizen has paid via ePay/EasyPay voucher and application is rejected

## Steps

1. Log in as administrator with payments.refund permission
2. Navigate to rejected application paid with voucher
3. Click 'Process Refund'
4. Verify refund method shows 'Credit Note'
5. Confirm refund amount matches original payment
6. Submit refund request
7. Verify credit note is generated with unique code
8. Verify credit note validity period is 12 months from issue date
9. Verify citizen receives email with credit note code and instructions

## Expected Result

Credit note is issued with 12-month validity. Credit note code is displayed (format: CN-YYYYMMDD-XXXXXX). Citizen can use credit note for any future service. Payment status changes to REFUNDED. Credit note details are stored in system.
