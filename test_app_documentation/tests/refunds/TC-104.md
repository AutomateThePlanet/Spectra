# Reject refund request exceeding original payment amount

## Preconditions

Application has been paid and is eligible for refund

## Steps

1. Log in as administrator with payments.refund permission
2. Navigate to application with payment (e.g., 50.00 BGN)
3. Select 'Process Refund'
4. Attempt to enter refund amount greater than original payment (e.g., 75.00 BGN)
5. Click 'Submit Refund'
6. Observe system validation

## Expected Result

System rejects the refund request with error message: 'Refund amount cannot exceed original payment amount of 50.00 BGN'. Refund is not processed. Payment status remains unchanged. No refund transaction is created.

## Test Data

Original payment: 50.00 BGN, Attempted refund: 75.00 BGN
