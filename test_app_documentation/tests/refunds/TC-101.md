# Refund bank transfer payment for cancelled service within 24 hours

## Preconditions

Citizen has paid via bank transfer and wants to cancel service within 24 hours of payment

## Steps

1. Log in as citizen who made bank transfer payment
2. Navigate to 'My Applications' dashboard
3. Select the paid application (within 24 hours of payment)
4. Click 'Cancel Application' button
5. System prompts for IBAN for refund
6. Enter valid IBAN (BG80BNBG96611020345678)
7. Confirm cancellation and refund request
8. Verify application status changes to 'Cancelled'
9. Verify refund request is created with status 'Pending'

## Expected Result

Refund is processed to the provided IBAN. System displays message 'Refund will be processed within 3-5 business days'. Citizen receives confirmation email with refund details. Payment status changes to REFUNDED.
