# Verify large dataset export triggers email notification

## Preconditions

Citizen account has dataset larger than 100 MB

## Steps

1. Create or use citizen account with data exceeding 100 MB
2. Initiate data export from My Data page
3. Set encryption password
4. Verify system displays message about email notification
5. Wait for processing to complete
6. Check citizen's email for export ready notification
7. Verify notification arrives within 24 hours

## Expected Result

For datasets larger than 100 MB, citizen receives an email notification when the export is ready (maximum processing time: 24 hours)

## Test Data

Citizen account with >100 MB of data
