# Verify data portability export in JSON format

## Preconditions

Citizen is logged into the portal

## Steps

1. Navigate to My Data page
2. Select data portability export option
3. Choose JSON format
4. Initiate export
5. Download the export file
6. Verify file is in JSON format
7. Verify JSON follows published schema
8. Verify includes: profile data, application history, documents metadata, payment history

## Expected Result

Export is provided in machine-readable JSON format following a published, versioned schema. Includes profile data, application history, documents metadata (not document files), and payment history

## Test Data

Citizen account with diverse data
