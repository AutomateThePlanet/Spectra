# Verify citizen can access personal data export via My Data portal

## Preconditions

Citizen is logged into the portal with an active account

## Steps

1. Navigate to portal settings
2. Click 'My Data' option
3. Review the data export options displayed
4. Verify the system shows what data will be included in the export

## Expected Result

System displays My Data page with options to export personal data including profile data, application history, documents, payment history, notification history, audit log entries, and consent records

## Test Data

Valid citizen account with existing data
