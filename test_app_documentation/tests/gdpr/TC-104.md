# Verify export file is encrypted with citizen-set password

## Preconditions

Citizen is ready to initiate data export

## Steps

1. Navigate to My Data page
2. Click to start data export
3. Enter a strong password when prompted
4. Confirm password
5. Complete export process
6. Download the export file
7. Attempt to open file without password
8. Verify file is encrypted
9. Open file with correct password
10. Verify successful decryption

## Expected Result

Export file is encrypted with the password set by the citizen before starting the export. File cannot be opened without the correct password
