# Verify citizen can update phone number with re-verification

## Preconditions

Citizen is logged into the portal

## Steps

1. Navigate to profile settings
2. Click to edit phone number
3. Enter new phone number
4. Submit the change
5. Check for SMS verification code
6. Enter verification code
7. Verify phone number is updated in profile

## Expected Result

Citizen can update phone number through self-service. New phone requires re-verification via SMS before the change is applied

## Test Data

Old phone: +359888123456, New phone: +359888654321
