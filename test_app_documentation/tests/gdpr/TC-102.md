# Verify data export excludes internal clerk notes

## Preconditions

Citizen has applications with internal clerk notes

## Steps

1. Ensure test citizen has at least one application with internal clerk notes
2. Initiate data export from My Data page
3. Set encryption password and complete export
4. Download and decrypt the export file
5. Search JSON and PDF for internal clerk notes
6. Verify no internal operational data is present

## Expected Result

Export does NOT include internal notes by clerks about the citizen's application, as these are considered internal operational data

## Test Data

Citizen account with applications containing internal clerk notes
