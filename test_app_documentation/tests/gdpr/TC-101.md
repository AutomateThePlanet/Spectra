# Verify data export includes all required data categories

## Preconditions

Citizen has completed data export request

## Steps

1. Initiate data export from My Data page
2. Set encryption password
3. Wait for export to complete
4. Download and decrypt the export file
5. Verify JSON and PDF files are present
6. Check JSON structure contains: profile data, application history, documents, payment history, notification history, audit log entries, consent records

## Expected Result

Export package contains structured JSON file and human-readable PDF summary with all required data categories: profile data, application history, documents, payment history, notification history, audit log entries about the citizen, and consent records

## Test Data

Citizen account with diverse data across all categories
