# Verify analytics cookies are off by default

## Preconditions

Cookie banner is displayed on first visit

## Steps

1. Display cookie banner
2. Click 'Customize' button
3. Check the state of 'Analytics Cookies' toggle
4. Verify toggle is OFF by default
5. Verify description mentions Google Analytics or Matomo with anonymized IP

## Expected Result

Analytics Cookies are off by default. Description indicates Google Analytics or Matomo with anonymized IP

## Test Data

First visit with no existing consent
