# Verify rectification retains previous values in version history

## Preconditions

Citizen has made changes to personal information

## Steps

1. Note current value of a personal data field (e.g., address)
2. Update the field to a new value
3. Confirm the change is saved
4. Access version history or audit log
5. Verify previous value is retained in history
6. Verify timestamp and change details are recorded

## Expected Result

Previous values are retained in the version history for audit purposes. Rectification does not delete historical records

## Test Data

Old address: 123 Main St, New address: 456 Oak Ave
