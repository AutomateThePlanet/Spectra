# Verify cookie preferences stored in first-party cookie

## Preconditions

User has set cookie preferences

## Steps

1. Display cookie banner
2. Set custom cookie preferences (e.g., accept analytics, reject functional)
3. Click 'Save Preferences'
4. Inspect browser cookies
5. Verify a first-party cookie is created with preference data
6. If user is authenticated, verify preferences are also stored in consent management database

## Expected Result

Cookie preferences are stored in a first-party cookie (not in the user's profile) and in the consent management database if the user is authenticated

## Test Data

Custom cookie preferences
