# Verify EGN correction requires administrator approval

## Preconditions

Citizen needs to correct EGN and has supporting document

## Steps

1. Navigate to profile settings
2. Request EGN correction
3. Upload official document from civil registry
4. Submit correction request
5. Verify request enters pending approval state
6. Administrator reviews and approves the request
7. Verify EGN is updated in profile
8. Check audit log for separate audit event

## Expected Result

EGN correction requires clerk assistance and administrator approval. Change is logged as a separate audit event

## Test Data

Supporting document: civil registry certificate
