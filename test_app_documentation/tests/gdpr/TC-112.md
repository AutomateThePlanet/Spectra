# Verify uploaded documents permanently deleted on account deletion

## Preconditions

Citizen has uploaded documents attached to applications

## Steps

1. Ensure citizen account has uploaded documents
2. Note document IDs and storage locations
3. Submit account deletion request
4. Wait for processing to complete
5. Verify uploaded documents are permanently deleted from storage
6. Verify system-generated documents are retained with anonymized citizen reference

## Expected Result

Documents uploaded by the citizen are permanently deleted. System-generated documents (certificates, decisions) are retained with anonymized citizen reference

## Test Data

Citizen account with uploaded identity documents and system-generated certificates
