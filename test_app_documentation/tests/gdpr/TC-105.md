# Verify export file auto-deletes after 7 days

## Preconditions

Citizen has completed a data export

## Steps

1. Complete data export and note the timestamp
2. Verify export file is available for download
3. Wait 7 days (or simulate time advancement in test environment)
4. Attempt to access the export file download link
5. Verify file is no longer available

## Expected Result

The export file is available for download for 7 days, then automatically deleted

## Test Data

Completed export file with known creation timestamp
