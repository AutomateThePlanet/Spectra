# Verify consent records include document version and timestamp

## Preconditions

Citizen is registering or updating consent

## Steps

1. Accept Terms of Service during registration
2. Accept Privacy Policy during registration
3. Access consent records (as administrator or through audit log)
4. Verify consent record includes: consent type, document version (e.g., Terms v2.3), timestamp, method (checkbox click), IP address, user agent

## Expected Result

Every consent action is recorded with: consent type, version of the document consented to (Terms v2.3, Privacy Policy v1.8), timestamp of consent/withdrawal, method (checkbox click, banner acceptance, API call), IP address and user agent at the time of consent

## Test Data

New citizen registration
