# Verify personal data anonymization on account deletion

## Preconditions

Citizen has no active applications

## Steps

1. Ensure citizen account has no active applications
2. Note current personal data values (name, EGN, email, phone, address)
3. Submit account deletion request
4. Wait for processing to complete (up to 30 days)
5. Verify personal data is anonymized: name → 'Citizen_XXXX', EGN → hash, email → deleted, phone → deleted, address → deleted
6. Verify application records are retained with anonymized citizen data

## Expected Result

Personal data is anonymized (not deleted): name → 'Citizen_A3F7', EGN → hash, email → deleted, phone → deleted, address → deleted. Application records are retained with anonymized citizen data

## Test Data

Citizen account with completed applications only
