# Verify cookie banner displays on first visit

## Preconditions

User has no existing consent cookie

## Steps

1. Clear all cookies and browser data
2. Navigate to the portal homepage
3. Verify cookie banner is displayed
4. Verify banner includes: Strictly Necessary Cookies (always active), Analytics Cookies (off by default), Functional Cookies (off by default)
5. Verify banner includes 'Accept All', 'Reject All', and 'Customize' buttons

## Expected Result

On first visit (no existing consent), the portal displays a cookie banner with three categories and three action buttons

## Test Data

Clean browser session
