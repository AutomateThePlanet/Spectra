# Verify account deletion blocked when active applications exist

## Preconditions

Citizen has at least one active application

## Steps

1. Ensure citizen account has active application(s)
2. Navigate to account settings
3. Click 'Delete Account' option
4. Confirm deletion request
5. Verify system displays error message

## Expected Result

System displays message: 'Your account cannot be deleted while you have active applications. Please wait for them to be completed or cancel them first.'

## Test Data

Citizen account with active application in 'pending' status
