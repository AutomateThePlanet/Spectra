# Verify data portability export in CSV format

## Preconditions

Citizen is logged into the portal

## Steps

1. Navigate to My Data page
2. Select data portability export option
3. Choose CSV format
4. Initiate export
5. Download the export file
6. Verify file is in CSV format
7. Verify includes: profile data, application history, documents metadata, payment history
8. Verify data is properly formatted in tabular structure

## Expected Result

Export is provided in machine-readable CSV format with tabular structure. Includes profile data, application history, documents metadata, and payment history

## Test Data

Citizen account with diverse data
