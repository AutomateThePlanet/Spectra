# Advanced filters with AND logic across multiple filter types

## Preconditions

User is authenticated and on the search results page with the advanced filters panel visible. Test data includes applications with various statuses, service types, and date ranges.

## Steps

1. Navigate to search results page
2. Open the advanced filters panel on the left side
3. Select 'Pending' from Status multi-select checkboxes
4. Select a specific service type from the Service Type dropdown
5. Set Date Range using date pickers: from '2026-01-01' to '2026-03-31'
6. Observe that results update immediately without clicking Apply button
7. Verify active filters are shown as chips above results
8. Verify URL query parameters reflect current filters

## Expected Result

Filters combine with AND logic (all conditions must be met). Results show only applications that are Pending AND match the selected service type AND fall within the date range. Results update within 300ms. Active filters appear as removable chips. URL is updated for bookmarking/sharing.

## Test Data

Status: Pending, Service Type: Building Permit, Date Range: 2026-01-01 to 2026-03-31
