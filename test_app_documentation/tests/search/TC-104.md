# Saved search creation exceeding maximum limit

## Preconditions

User is authenticated and already has 20 saved searches (maximum limit). User is on a search results page.

## Steps

1. Verify user already has 20 saved searches
2. Execute a new search query with specific filters
3. Click 'Save this search' button on the results page
4. Attempt to enter a name for the new saved search
5. Observe system behavior when trying to save the 21st search

## Expected Result

System prevents saving more than 20 searches per user. An error message is displayed indicating the maximum limit has been reached. User is prompted to delete an existing saved search before creating a new one.

## Test Data

User with 20 existing saved searches, attempting to save search: 'status:pending region:Sofia'
