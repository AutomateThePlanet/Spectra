# Quick search autocomplete with minimum 3 characters

## Preconditions

User is authenticated and on any page with the global search bar in top navigation

## Steps

1. Locate the global search bar in the top navigation
2. Type 2 characters into the search field
3. Observe that no autocomplete suggestions appear
4. Type a third character (total 3 characters)
5. Wait 300ms for debounce
6. Observe the autocomplete dropdown

## Expected Result

Autocomplete suggestions appear after typing 3 or more characters with a 300ms debounce. Suggestions are grouped by category (Applications, Citizens, Documents) with maximum 5 suggestions per category (15 total). Each suggestion shows title/name, category icon, and brief context line. Response time is under 200ms.

## Test Data

Search query: 'per' (3 characters)
