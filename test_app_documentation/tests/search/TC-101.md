# Search using exact phrase syntax with quotes

## Preconditions

User is authenticated with access to search functionality. Test data includes applications with 'building permit' as exact phrase and applications with 'building' and 'permit' as separate words.

## Steps

1. Navigate to the global search bar
2. Enter search query: "building permit" (with quotes)
3. Press Enter to execute search
4. Review search results on the results page
5. Verify that only results containing the exact phrase 'building permit' are returned
6. Verify that results with 'building' and 'permit' as separate words are excluded

## Expected Result

Search returns only results containing the exact phrase 'building permit' as specified in the query syntax. Results are displayed within 500ms for queries matching up to 100,000 documents.
