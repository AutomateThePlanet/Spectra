# Refund processing for rejected application

## Preconditions

Citizen has submitted and paid for a service application using a bank card. Application has been rejected by administration.

## Steps

1. Submit a service application and complete card payment (e.g., Criminal Record Certificate, 5.00 BGN)
2. Verify payment status is PAID and application is IN_PROCESSING
3. Administrator rejects the application with a reason
4. Verify refund is automatically triggered
5. Verify refund is processed back to the original card
6. Verify refund amount equals the original payment amount (5.00 BGN)
7. Verify citizen is notified of refund via email
8. Verify refund processing time is stated as 5-10 business days
9. Verify refund transaction is logged in audit trail

## Expected Result

When application is rejected, refund is automatically processed. For card payments, refund goes back to the original card. Refund amount equals original payment. Citizen receives email notification. Processing time is 5-10 business days. Transaction is fully audited.

## Test Data

Service: Criminal Record Certificate, Fee: 5.00 BGN, Payment Method: Visa ending in 5678, Rejection Reason: Incomplete documentation
