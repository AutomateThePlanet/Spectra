# Bank transfer payment with exact amount match

## Preconditions

Citizen has submitted a service application requiring payment. Bank transfer is selected as payment method.

## Steps

1. Submit a service application (e.g., Address Registration, 3.00 BGN)
2. Select 'Bank Transfer' as payment method
3. Verify system generates unique payment reference in format PR-YYYYMMDD-XXXXXX
4. Verify payment reference and bank details are displayed on screen
5. Verify confirmation email is sent with payment reference and bank details
6. Simulate bank transfer arrival with exact amount (3.00 BGN) and correct payment reference
7. Verify payment is matched automatically
8. Verify payment status changes to PAID
9. Verify application status changes to IN_PROCESSING
10. Verify receipt is generated and emailed to citizen

## Expected Result

System generates unique payment reference (PR-YYYYMMDD-XXXXXX). Payment reference and bank details are displayed and emailed. When bank transfer arrives with exact amount and correct reference, payment is automatically matched. Application proceeds to IN_PROCESSING status. Receipt is generated.

## Test Data

Service: Address Registration, Fee: 3.00 BGN, Payment Reference: PR-20260320-123456
