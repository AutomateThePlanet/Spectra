# Successful card payment with 3D Secure authentication

## Preconditions

Citizen has submitted a service application with a fee of at least 0.50 BGN. Citizen has a valid Visa or Mastercard.

## Steps

1. Navigate to the payment page for the submitted application
2. Select 'Bank Card (Visa/Mastercard)' as the payment method
3. Click 'Proceed to Payment'
4. Verify redirect to ePay.bg payment gateway
5. Enter valid card details (card number, expiry, CVV)
6. Complete 3D Secure authentication (enter OTP or use biometric)
7. Verify callback redirects back to the system
8. Check payment status changes to PAID
9. Verify application status changes to IN_PROCESSING
10. Verify receipt PDF is generated and available for download
11. Verify confirmation email is sent with receipt attached

## Expected Result

Payment is successfully processed. Application status changes to IN_PROCESSING. Receipt is generated with unique receipt number (R-YYYY-XXXXXX), contains masked card number (last 4 digits only), digital signature, and all required fields. Citizen receives confirmation email with PDF receipt.

## Test Data

Service: Certificate of Birth, Fee: 5.00 BGN, Card: Visa ending in 1234
