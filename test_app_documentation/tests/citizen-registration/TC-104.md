# Attempt registration with duplicate EGN for active account

## Preconditions

An active citizen account already exists with EGN '8001011234' (example valid EGN)

## Steps

1. Navigate to the citizen registration portal
2. Complete Steps 1-5 of registration wizard using the same EGN as existing active account ('8001011234')
3. Step 6: Review information and click 'Confirm'
4. Observe duplicate detection check before account creation
5. Verify error message is displayed

## Expected Result

System detects duplicate EGN and displays message: 'An account with this identification number already exists. Please use the password reset function if you cannot access your account.' Account creation is blocked.

## Test Data

Duplicate EGN: 8001011234 (matches existing active account)
