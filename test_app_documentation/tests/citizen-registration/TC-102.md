# Attempt registration with invalid EGN checksum

## Preconditions

None

## Steps

1. Navigate to the citizen registration portal
2. Step 1: Enter personal information with invalid EGN (10 digits but incorrect checksum, e.g., '1234567890')
3. Tab out of the EGN field to trigger on-blur validation
4. Observe validation error message
5. Click 'Next' button
6. Verify validation error is displayed and highlighted in red

## Expected Result

EGN field is highlighted in red with error message 'Invalid EGN checksum' or similar. User cannot proceed to Step 2 until a valid EGN is entered. Next button shows all validation errors on click.

## Test Data

Invalid EGN: 1234567890 (fails Bulgarian EGN checksum algorithm)
