# Complete citizen registration with valid Bulgarian EGN and QES verification

## Preconditions

User has a valid Bulgarian eID or qualified certificate for QES signing

## Steps

1. Navigate to the citizen registration portal
2. Step 1: Enter valid personal information (First Name: 'Иван', Last Name: 'Иванов', DOB: 15/03/1990, EGN: valid 10-digit with correct checksum, Gender: auto-filled from EGN, Nationality: Bulgarian)
3. Click 'Next' and verify no validation errors appear
4. Step 2: Enter email address and mobile phone (+359 888 123 456)
5. Enter the 6-digit verification code received via email
6. Enter the 6-digit verification code received via SMS
7. Click 'Next' after both verifications are complete
8. Step 3: Select structured address (Region: София-град, Municipality: Столична, City: София, Postal Code: 1000, Street: 'ул. Витоша', Number: '1')
9. Click 'Next'
10. Step 4: Select 'Qualified Electronic Signature (QES/КЕП)' as identity verification method
11. Sign the verification challenge using Bulgarian eID
12. Wait for real-time verification to complete successfully
13. Click 'Next'
14. Step 5: Enter password meeting policy requirements, check 'Terms and Conditions' checkbox
15. Click 'Next'
16. Step 6: Review all information on confirmation page
17. Click 'Confirm' to complete registration
18. Verify welcome email is received
19. Verify redirect to citizen dashboard

## Expected Result

Account is created successfully, welcome email is sent without password, citizen is redirected to dashboard with full access to submit applications
