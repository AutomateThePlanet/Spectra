# Register with email verification code expiration and retry

## Preconditions

None

## Steps

1. Complete Step 1 of registration wizard with valid data
2. Step 2: Enter valid email address
3. Receive 6-digit verification code via email
4. Wait for 16 minutes (code expires after 15 minutes)
5. Attempt to enter the expired verification code
6. Observe error message indicating code has expired
7. Click 'Request new code' button
8. Receive new 6-digit verification code via email
9. Enter the new code within 15 minutes
10. Verify email is marked as verified
11. Complete phone verification
12. Proceed to Step 3

## Expected Result

Expired code is rejected with clear error message. User can request a new code (up to 3 times per hour). New code is sent and accepted when entered within 15-minute window. Email verification status is preserved.
