# TOTP MFA code successfully validates login

## Preconditions

User has TOTP MFA enrolled (Google Authenticator or Microsoft Authenticator)

## Steps

1. Login with valid username and password
2. System prompts for MFA code
3. Open authenticator app and retrieve current TOTP code
4. Enter the 6-digit TOTP code
5. Submit the code

## Expected Result

TOTP code is validated successfully. User is authenticated and redirected to dashboard. MFA_SUCCESS event is logged to audit trail.

## Test Data

Valid TOTP code from authenticator app
