# SMS OTP code sent and valid for 5 minutes

## Preconditions

User has SMS OTP enrolled with registered phone number

## Steps

1. Login with valid username and password
2. Select SMS OTP as MFA method
3. System sends 6-digit code to registered phone
4. Receive SMS with code
5. Enter the code within 5 minutes
6. Submit the code

## Expected Result

SMS OTP code is sent to registered phone number. Code is valid for 5 minutes. User is authenticated when correct code is entered within validity period. MFA_SUCCESS event is logged.

## Test Data

Valid SMS OTP code received within 5 minutes
