# MFA session invalidated after 3 failed code attempts

## Preconditions

User has completed username/password authentication and is at MFA prompt

## Steps

1. Login with valid username and password
2. System prompts for MFA code
3. Enter incorrect MFA code (1st attempt)
4. Enter incorrect MFA code (2nd attempt)
5. Enter incorrect MFA code (3rd attempt)
6. Observe system behavior

## Expected Result

After 3 failed MFA code attempts, the MFA session is invalidated. User must restart the login process from the beginning (username/password). MFA_FAILED events are logged for each failed attempt.

## Test Data

Incorrect MFA codes
