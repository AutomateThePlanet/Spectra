# Password reset link can only be used once

## Preconditions

User has requested password reset and received reset link

## Steps

1. Request password reset and receive email with reset link
2. Click the reset link
3. Set new password successfully
4. Attempt to use the same reset link again

## Expected Result

Password reset link can be used exactly once. After successful password change, the link is invalidated. Second attempt to use the same link fails with error message indicating link has already been used or is invalid.
