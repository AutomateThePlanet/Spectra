# Login fails with incorrect password

## Preconditions

User account exists

## Steps

1. Navigate to the login page
2. Enter valid email address
3. Enter incorrect password
4. Click the Login button

## Expected Result

Login fails with error message. LOGIN_FAILED event is logged to audit trail with failure reason. Failed attempt counter increments toward account lockout threshold.

## Test Data

Valid email, incorrect password
