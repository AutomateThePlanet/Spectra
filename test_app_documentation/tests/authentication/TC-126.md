# All active sessions terminated after password change

## Preconditions

User has multiple active sessions

## Steps

1. Login from Browser 1 and establish active session
2. Login from Browser 2 and establish active session
3. Verify both sessions are active
4. From Browser 1, change password via password reset flow
5. Attempt to perform action in Browser 2

## Expected Result

All active sessions for the user are terminated upon password change. User in Browser 2 is logged out and must re-authenticate with new password. SESSION_EXPIRED events are logged for all terminated sessions.

## Test Data

User with multiple active sessions
