# Password rejected when less than 12 characters

## Preconditions

User is setting or changing password

## Steps

1. Navigate to password change form
2. Enter password with only 11 characters
3. Include uppercase, lowercase, digit, and special character
4. Submit the password

## Expected Result

Password is rejected with error message indicating minimum 12 character requirement. Password is not saved.

## Test Data

Example: MyP@ss0rd1 (11 chars)
