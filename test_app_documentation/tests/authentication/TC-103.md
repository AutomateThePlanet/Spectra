# Account lockout after 5 consecutive failed login attempts

## Preconditions

User account exists and is not locked

## Steps

1. Navigate to the login page
2. Enter valid email address
3. Enter incorrect password and click Login (repeat 5 times)
4. Observe the response after the 5th failed attempt

## Expected Result

After the 5th failed attempt, account is locked for 30 minutes. User sees message: 'Your account has been temporarily locked. Please try again later or contact support.' ACCOUNT_LOCKED event is logged to audit trail.

## Test Data

Valid email, incorrect password
