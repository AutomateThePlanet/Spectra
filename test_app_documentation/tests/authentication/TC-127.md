# Rate limiting blocks IP after 10 login attempts per minute

## Preconditions

No existing rate limit blocks on test IP address

## Steps

1. From a single IP address, attempt to login 10 times within 1 minute
2. Attempt an 11th login from the same IP address
3. Observe the response

## Expected Result

After 10 login attempts per minute from the same IP, the 11th attempt is blocked with HTTP 429 response. IP is blocked for 1 hour. Error message indicates rate limit exceeded.

## Test Data

Single IP address, any credentials
