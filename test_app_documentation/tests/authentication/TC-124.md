# Self-service password reset link sent and valid for 1 hour

## Preconditions

User account exists with registered email

## Steps

1. Navigate to login page
2. Click 'Forgot Password' link
3. Enter registered email address
4. Submit the form
5. Check email for password reset link
6. Click the reset link within 1 hour
7. Set new password meeting policy requirements

## Expected Result

Password reset link is sent to registered email and is valid for 1 hour. User can set new password within validity period. PASSWORD_RESET_REQUESTED event is logged when link is requested. PASSWORD_CHANGED event is logged when password is successfully changed.

## Test Data

Registered email address, new password meeting policy
