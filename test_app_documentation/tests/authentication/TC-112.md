# MFA enrollment forced for internal user without MFA

## Preconditions

Internal user account exists without MFA enrolled, MFA is mandatory for internal users

## Steps

1. Login as internal user with valid username and password
2. User has not enrolled in any MFA method
3. Complete username/password authentication

## Expected Result

System forces MFA enrollment before granting access. User is presented with MFA setup options (TOTP, SMS OTP, Email OTP) and cannot proceed to dashboard until enrollment is complete.

## Test Data

Internal user without MFA
