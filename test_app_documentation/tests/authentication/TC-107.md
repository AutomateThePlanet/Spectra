# Password rejected when missing uppercase letter

## Preconditions

User is setting or changing password

## Steps

1. Navigate to password change form
2. Enter password with 12+ characters
3. Include lowercase letters, digits, and special characters
4. Do NOT include any uppercase letters
5. Submit the password

## Expected Result

Password is rejected with error message indicating uppercase letter requirement. Password is not saved.

## Test Data

Example: myp@ssw0rd123 (no uppercase)
