# Temporary password expires after 24 hours

## Preconditions

Administrator issued temporary password to user

## Steps

1. Administrator issues temporary password for user
2. Wait 24 hours and 1 minute
3. Attempt to login with the temporary password

## Expected Result

Login fails with message indicating temporary password has expired. User must request a new temporary password or use password reset flow.

## Test Data

Temporary password issued 24+ hours ago
