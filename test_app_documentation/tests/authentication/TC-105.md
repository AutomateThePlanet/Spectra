# Administrator manually unlocks locked account

## Preconditions

User account is locked, administrator has unlock permissions

## Steps

1. Lock user account by failing login 5 times
2. Login as administrator
3. Navigate to user management
4. Select the locked user account
5. Click 'Unlock Account' button
6. Attempt to login as the unlocked user with correct credentials

## Expected Result

Administrator successfully unlocks the account before 30-minute window expires. User can immediately login with correct credentials. ACCOUNT_UNLOCKED event is logged with administrator identifier.

## Test Data

Locked user account, admin credentials
