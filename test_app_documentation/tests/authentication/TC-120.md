# Session timeout after 60 minutes of inactivity for citizens

## Preconditions

Citizen user is logged in

## Steps

1. Login as citizen user
2. Perform an action to establish active session
3. Do not interact with the system for 60 minutes
4. After 60 minutes, attempt to perform an action

## Expected Result

After 60 minutes of inactivity, session expires. User is redirected to login page. SESSION_EXPIRED event is logged to audit trail.

## Test Data

Citizen user session
