# Fourth concurrent session terminates oldest session

## Preconditions

User account exists

## Steps

1. Login from Browser 1 (Session 1)
2. Login from Browser 2 (Session 2)
3. Login from Browser 3 (Session 3)
4. Verify all 3 sessions are active
5. Login from Browser 4 (Session 4)
6. Attempt to perform action in Browser 1 (oldest session)

## Expected Result

Maximum 3 concurrent sessions are allowed. When 4th session is initiated, the oldest session (Session 1) is terminated automatically. User in Browser 1 is logged out and must re-authenticate.

## Test Data

Single user account, 4 different browsers/devices
