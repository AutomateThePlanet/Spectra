# Temporary password must be changed on first login

## Preconditions

Administrator issued temporary password to user

## Steps

1. Administrator issues temporary password for user
2. User receives temporary password via email
3. User logs in with temporary password
4. Observe system behavior

## Expected Result

System forces user to change password before granting access. User cannot proceed to dashboard until new password is set. New password must meet password policy requirements.

## Test Data

Valid temporary password
