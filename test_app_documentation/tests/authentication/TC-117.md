# SSO authentication via Azure AD SAML 2.0

## Preconditions

SSO is enabled for user's domain, Azure AD tenant is configured

## Steps

1. Navigate to login page
2. Enter email address with SSO-enabled domain
3. Observe automatic redirect to Azure AD IdP
4. Authenticate with Azure AD credentials
5. Complete Azure AD authentication flow
6. Observe redirect back to application

## Expected Result

Username/password form is hidden. User is automatically redirected to Azure AD IdP. After successful Azure AD authentication, user is redirected back and authenticated in the application. LOGIN_SUCCESS event is logged.

## Test Data

User with SSO-enabled domain, valid Azure AD credentials
