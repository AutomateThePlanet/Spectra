# Session timeout after 30 minutes of inactivity for internal users

## Preconditions

Internal user is logged in

## Steps

1. Login as internal user
2. Perform an action to establish active session
3. Do not interact with the system for 30 minutes
4. After 30 minutes, attempt to perform an action

## Expected Result

After 30 minutes of inactivity, session expires. User is redirected to login page. SESSION_EXPIRED event is logged to audit trail.

## Test Data

Internal user session
