# Password meets minimum complexity requirements

## Preconditions

User is setting or changing password

## Steps

1. Navigate to password change form
2. Enter password with minimum 12 characters
3. Include at least one uppercase letter (A-Z)
4. Include at least one lowercase letter (a-z)
5. Include at least one digit (0-9)
6. Include at least one special character (!@#$%^&*)
7. Submit the password

## Expected Result

Password is accepted and saved. PASSWORD_CHANGED event is logged to audit trail.

## Test Data

Example: MyP@ssw0rd123
