# Session invalidated when IP address changes mid-session

## Preconditions

User is logged in with active session

## Steps

1. Login and establish active session from IP address A
2. Perform an action to verify session is active
3. Change IP address (e.g., switch networks, use VPN)
4. Attempt to perform another action from IP address B

## Expected Result

Session is invalidated when IP address changes mid-session. User is logged out and must re-authenticate. SESSION_EXPIRED event is logged with IP change noted.

## Test Data

Active session, ability to change IP address
