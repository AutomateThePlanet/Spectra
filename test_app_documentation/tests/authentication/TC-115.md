# Email OTP code sent and valid for 10 minutes

## Preconditions

User has Email OTP enrolled with registered email

## Steps

1. Login with valid username and password
2. Select Email OTP as MFA method
3. System sends 6-digit code to registered email
4. Check email and retrieve code
5. Enter the code within 10 minutes
6. Submit the code

## Expected Result

Email OTP code is sent to registered email address. Code is valid for 10 minutes. User is authenticated when correct code is entered within validity period. MFA_SUCCESS event is logged.

## Test Data

Valid Email OTP code received within 10 minutes
