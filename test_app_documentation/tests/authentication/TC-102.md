# Login fails with non-existent username

## Preconditions

Email address does not exist in the system

## Steps

1. Navigate to the login page
2. Enter non-existent email address
3. Enter any password
4. Click the Login button

## Expected Result

Login fails with generic error message that does not reveal whether the account exists. LOGIN_FAILED event is logged with user identifier as 'unknown'.

## Test Data

Non-existent email address
